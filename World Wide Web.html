World Wide Web

The World Wide Web (WWW) is a vast system of interlinked hypertext documents and multimedia content accessible via the Internet. It was invented by British scientist Tim Berners-Lee in 1989 while working at CERN, and became publicly available in 1991. The Web is not the same as the Internet — the Internet is the underlying global network infrastructure, while the Web is one of many services that runs on top of it..

Web content is stored on web servers and accessed by users through web browsers using URLs. Documents on the Web are written in HTML and can link to one another through hyperlinks, creating a web-like structure of connected information spanning the entire globe.

  • - Created by Tim Berners-Lee in 1989; publicly launched in 1991
  • - Built on three core technologies: HTML, HTTP, and URLs
  • - The Web is a service on the Internet, not the Internet itself
