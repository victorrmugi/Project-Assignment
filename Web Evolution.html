Web Evolution

Web 1.0

Web 1.0 refers to the first generation of the World Wide Web, spanning roughly from its inception in 1991 to around 2004. It is characterised as the "read-only" web — a static, one-directional medium where website owners created content and users simply consumed it without any means of interacting or contributing. Websites in this era consisted primarily of static HTML pages with basic styling. There were no user accounts, comment sections, or user-generated content. Navigation was via hyperlinks, and web pages were largely informational — functioning like online brochures or digital encyclopaedias. Examples include early corporate websites, directories like Yahoo!, and the original versions of many news sites.

Web 2.0

Web 2.0 describes the second generation of the Web, emerging around 2004–2005 and continuing to the present. It is characterised as the "read-write" web — a participatory, social, and dynamic web where users are no longer passive consumers but active creators and contributors of content. The term was popularised by Tim O'Reilly in 2004. Web 2.0 is defined by user-generated content (blogs, wikis, reviews), social networking (Facebook, Twitter/X, Instagram), rich interactive applications (Google Maps, Gmail), and cloud-based platforms. Technologies such as AJAX, JavaScript frameworks, REST APIs, and the rise of smartphones fuelled this transformation. Data became a key asset, and platforms thrived on network effects — growing more valuable as more users joined.

Web 3.0

Web 3.0 represents the third generation of the Web, which is still in development. It is characterised as the "semantic" web — a decentralised, intelligent web where data is interconnected and machines can understand and process it more effectively. The goal is to create a more personalised, efficient, and user-centric web experience. Web 3.0 leverages technologies such as blockchain, artificial intelligence, and the semantic web to enable more sophisticated interactions and data management. It aims to provide users with greater control over their data and a more seamless integration of online and offline experiences.

Web 4.0

Web 4.0 is the anticipated fourth generation of the Web, sometimes called the "Symbiotic Web" or "Intelligent Web." It envisions a highly intelligent, proactive, and seamlessly connected digital environment where AI, the Internet of Things (IoT), augmented/virtual reality, and ambient computing converge to create a web that understands human intent and acts autonomously on users' behalf. In Web 4.0, the distinction between the physical and digital worlds blurs. Intelligent agents and AI systems anticipate user needs, devices communicate autonomously, and the web becomes a pervasive, ambient presence embedded in everyday life — from smart cities to connected healthcare. The European Union released a Web 4.0 strategy in 2023, emphasising open standards, virtual worlds, and digital-physical integration. While still largely conceptual and emerging, Web 4.0 represents the direction in which the web is evolving.
